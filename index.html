🔗 Коннектор файлов

Проект демонстрирует подключение нескольких JS файлов и работу с их данными

Нажмите "Загрузить данные", чтобы увидеть результат работы JavaScript

📝 Что происходит:

  1. Загружается data.js с массивом пользователей
  2. Загружается utils/*.js С логикой приложения
  3. Загружаются исторические модули: historical/*.js
  4. Загружаются statistics.js и ui.js
  5. app.js инициализирует приложение
